Курсовой проект по БД

Цель провекта.
Разработать базу данных для онлайн-площадки по размещению, поиску и краткосрочной аренде частного жилья по всему миру, по примеру веб-сервиса https://www.airbnb.ru/

Название БД rental_housing

Ресурс работает следующим образом. На сайте регистрируются пользователи которые предлагают в аренду жилье и пользователи которые ищут подходящее для аренды жильё. 
Пользователи предлагающие жилье могут размещать несколько позиций сдаваемого жилья и для каждой позиции делать описание и сопроводать фотографиями.
Пользователь который арендует жильё может вступить в переписку с хозяином предлогающим жилье.
Если обе стороны сделки договорились то заключается сделка.
После окончания сделки каждый из сторон может оставить свой отзыв, соответственно о сдаваемом жилье или проживающих клиентах.
Так же добавлена система лайков для сообщений об отзывах, сдаваемом жилье, клиентах 

В структуру БД входят следующие таблицы:

users - таблица зарегестрированных пользователей

profiles - таблица профиля пользователя

countries - таблица стран

addresses - таблица адресов: 
    для жилья которое сдается в аренду
    для регистрации места проживания пользователя

orders - таблица сделок по сдаче жилья в аренду

rents - таблица сдаваемого в аренду жилья

photos - таблица фотографий. Хранится в виде пути к файлу. Там могут хранится 
    1. аватарка пользователя
    2. фотографии сдаваемого в аренду жилья

photos_rent - таблица связей для фотографий арендуемого жилья.

messages - таблица переписок между продавцом и покупателем

reviews - таблица отзывов по сдаваемому жилью и клиентам

likes, target_types  - таблица лайков
